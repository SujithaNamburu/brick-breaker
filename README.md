
This is a simple Brick Breaker game developed using Java. The game is built with Java Swing and follows basic object-oriented programming concepts.

Project Description The player controls a paddle to bounce the ball and break the bricks. The goal is to break all the bricks without letting the ball fall below the paddle.

Features Paddle movement using keyboard Ball collision with walls, paddle, and bricks Score display Game over and win condition Restart option Technologies Used Java Java Swing AWT Controls Left Arrow Key – Move paddle left Right Arrow Key – Move paddle right Enter Key – Restart the game
